Sistema de Inventario para Pequeños Negocios
1. Descripción del Proyecto

Sistema de gestión de inventario desarrollado en Java que permitirá a pequeños negocios administrar productos, controlar existencias, gestionar múltiples almacenes o sucursales y registrar movimientos de inventario.

El sistema incluye:

* Gestión de usuarios

* Control de inventario

* Registro de entradas y salidas

* Transferencias entre almacenes

* Autenticación de usuarios

4.1 Módulo de Autenticación
